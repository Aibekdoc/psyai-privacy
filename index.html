Политика конфиденциальности PSY AI Coach

Последнее обновление: 6 мая 2026 г.

1. Сбор данных

PSY AI Coach не собирает и не хранит личные данные пользователей на собственных серверах. История диалогов хранится только локально на устройстве пользователя.

2. Передача данных третьим сторонам

Для работы приложения данные передаются следующим третьим сторонам:

  • Anthropic Claude API — текстовые сообщения отправляются для генерации ответов AI-коуча. Данные не хранятся и не используются для обучения моделей.
  • OpenAI Whisper API — голосовые записи отправляются для распознавания речи и немедленно удаляются после обработки.
  • RevenueCat — данные о подписках для обработки платежей.

3. Согласие пользователя

При первом запуске приложение явно запрашивает согласие пользователя на передачу данных указанным третьим сторонам. Без согласия использование приложения невозможно.

4. Голосовые данные

Голосовые записи обрабатываются через OpenAI Whisper API и немедленно удаляются после транскрипции.

5. Подписки и платежи

Платёжные данные обрабатываются через Apple App Store и RevenueCat. Мы не имеем доступа к платёжной информации.

6. Права пользователей

Пользователи могут удалить все локальные данные через функцию "Сбросить и начать заново" в настройках приложения.
