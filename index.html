Политика конфиденциальности PSY AI Coach

Последнее обновление: 27 апреля 2026 г.

1. Сбор данных

PSY AI Coach не собирает и не хранит личные данные пользователей на внешних серверах. Все данные хранятся локально на устройстве.

2. Голосовые данные

Голосовые записи обрабатываются через OpenAI Whisper API и немедленно удаляются после обработки.

3. AI-обработка

Сообщения обрабатываются через Anthropic Claude API. Данные не используются для обучения моделей.

4. Подписки

Платёжные данные обрабатываются через Apple App Store. Мы не имеем доступа к платёжной информации.
